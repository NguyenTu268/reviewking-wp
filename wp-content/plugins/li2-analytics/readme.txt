LI2 tracking integration for WooCommerce.

== Description ==

LI2 Analytics is a tracking integration for WooCommerce. It automatically tracks page views, clicks, and scrolling events using the LI2 Analytics client script, and securely tracks WooCommerce orders via server-to-server webhook events.

== Installation ==

1. Upload the `li2-analytics` plugin directory to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > LI2 Analytics** (or **LI2 Analytics** top-level menu) in the sidebar.
4. Enter your `Publishable Key` and `Server API Key` provided by LI2.
5. Save the settings.

== Frequently Asked Questions ==
